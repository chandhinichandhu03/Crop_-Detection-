## Chilli (Hot Pepper)

**Scientific Name**: *Capsicum annuum* L.
**IUCN Conservation Status**: `LC`

### 📋 Botanical Taxonomy Hierarchy

| Rank | Name | Description |
| --- | --- | --- |
| **Kingdom** | Plantae | Plant kingdom |

| **Order** | Solanales | Order grouping |
| **Family** | Solanaceae | Nightshade family; tomato, potato, pepper, tobacco |
| **Genus** | Capsicum | Genus grouping |
