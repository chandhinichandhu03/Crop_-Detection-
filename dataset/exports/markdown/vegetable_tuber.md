## Cassava (Tapioca)

**Scientific Name**: *Manihot esculenta* Crantz
**IUCN Conservation Status**: `LC`

### 📋 Botanical Taxonomy Hierarchy

| Rank | Name | Description |
| --- | --- | --- |
| **Kingdom** | Plantae | Plant kingdom |
| **Division** | Tracheophyta | Vascular/Non-vascular division |
| **Class** | Magnoliopsida | Class classification |
| **Order** | Malpighiales | Order grouping |
| **Family** | Euphorbiaceae | Spurge family; includes cassava, rubber, castor |
| **Genus** | Manihot | Genus grouping |
| **Species** | *esculenta* | Species epithet |
