## Banyan Tree

**Scientific Name**: *Ficus benghalensis* L.
**IUCN Conservation Status**: `LC`

### 📋 Botanical Taxonomy Hierarchy

| Rank | Name | Description |
| --- | --- | --- |
| **Kingdom** | Plantae | Plant kingdom |
| **Division** | Tracheophyta | Vascular/Non-vascular division |
| **Class** | Magnoliopsida | Class classification |
| **Order** | Rosales | Order grouping |
| **Family** | Moraceae | Mulberry family; includes fig, breadfruit, jackfruit |
| **Genus** | Ficus | Genus grouping |
| **Species** | *benghalensis* | Species epithet |
