## Muskmelon / Cantaloupe

**Scientific Name**: *Cucumis melo* L.
**IUCN Conservation Status**: `LC`

| Rank | Name | Description |
| --- | --- | --- |
| **Kingdom** | Plantae | Plant kingdom |
| **Division** | Tracheophyta | Vascular/Non-vascular division |
| **Class** | Magnoliopsida | Class classification |
| **Order** | Cucurbitales | Order grouping |
| **Family** | Cucurbitaceae | Gourd/Squash family; cucumbers, melons, pumpkins |
| **Genus** | Cucumis | Genus grouping |
| **Species** | *melo* | Species epithet |
