## Groundnut (Peanut)

**Scientific Name**: *Arachis hypogaea* L.
**IUCN Conservation Status**: `LC`

### 📋 Botanical Taxonomy Hierarchy

| Rank | Name | Description |
| --- | --- | --- |
| **Kingdom** | Plantae | Plant kingdom |
| **Division** | Tracheophyta | Vascular/Non-vascular division |
| **Class** | Magnoliopsida | Class classification |
| **Order** | Fabales | Order grouping |
| **Family** | Fabaceae | Legume/Pea family; world's second largest food plant family |
| **Genus** | Arachis | Genus grouping |
| **Species** | *hypogaea* | Species epithet |
